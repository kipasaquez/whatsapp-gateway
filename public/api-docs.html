WhatsApp Gateway API

Dashboard

Send Message API

Method: POST
Endpoint: /send-text
Headers: 
Content-Type: application/json
Request Body: 
{
    "number": "6281234567890",  // Recipient number (required)
    "message": "Hello World!"    // Message text (required)
}
Response: 
{
    "status": true,
    "message": "Pesan teks berhasil dikirim.",
    "messageId": "XXXXXXXXXXXXX"
}

Send Media API

Method: POST
Endpoint: /send-media
Headers: 
Content-Type: application/json
Request Body: 
{
    "number": "6281234567890",           // Recipient number (required)
    "type": "image",                     // Media type: "image", "video", "document" (required)
    "url": "https://example.com/img.jpg", // Media URL (required)
    "caption": "Image caption"           // Caption text (optional)
}
Response: 
{
    "status": true,
    "message": "Pesan media berhasil dikirim."
}

Get Status API

Method: GET
Endpoint: /api/status
Response: 
{
    "connected": true,
    "uptime": "1h 30m 45s",
    "messagesSent": 42
}

Example Usage (cURL)

Send Text Message: 
curl -X POST http://localhost:8000/send-text \
  -H "Content-Type: application/json" \
  -d '{
    "number": "6281234567890",
    "message": "Hello World!"
  }'
Send Media Message: 
curl -X POST http://localhost:8000/send-media \
  -H "Content-Type: application/json" \
  -d '{
    "number": "6281234567890",
    "type": "image",
    "url": "https://example.com/image.jpg",
    "caption": "Check this out!"
  }'
Get Status: 
curl http://localhost:8000/api/status
